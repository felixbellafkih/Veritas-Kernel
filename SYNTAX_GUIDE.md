# GUIDE DE SYNTAXE VERITAS (VTS-v3)

## 1. VOYELLES CONSONANTIQUES (INPUT)
* **A** = Alif / Hamza (أ / إ / آ)
* **'** = 'Ain (ع)

## 2. CONSONNES EMPHATIQUES (HARDWARE)
Marquées par un point `.` après la lettre.

| Lettre Arabe | Code VTS | Concept | Exemple |
| :--- | :--- | :--- | :--- |
| **S** (س) | `S` | Software / Flux | S-B-L |
| **Ṣ** (ص) | `S.` | Hardware / Structure | S.-R-T. |
| **T** (ت) | `T` | Instruction | T-W-B |
| **Ṭ** (ط) | `T.` | Hardware / Percussion | T.-R-Q |
| **D** (د) | `D` | Pointeur | D-L-L |
| **Ḍ** (ض) | `D.` | Compression | D.-L-L |
| **Ẓ** (ظ) | `Z.` | Entropie | Z.-L-M |
